# Binance Futures Trading Bot

## Overview
This project is a Python CLI trading bot that places MARKET and LIMIT orders on Binance Futures Testnet.

## Features
- Place MARKET orders
- Place LIMIT orders
- BUY and SELL support
- CLI input validation
- Logging of API responses
- Error handling
